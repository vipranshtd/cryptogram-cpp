# Crypt-O-Gram (C++)

A console-based cryptography program implemented in C++ that supports
Morse Code and Caesar Cipher encryption and decryption.

## Features
- Morse Code encoder and decoder
- Caesar Cipher encoder and decoder
- Automatic Caesar Cipher decoding using word matching
- File input/output support
- Menu driven console interface

## Technologies
- C++
- File Handling
- STL (map, vector, string)

## How to Run

1. Compile the program

g++ cryptogram.cpp -o cryptogram

2. Run the program

./cryptogram

## Example Capabilities
- Encode text into Morse Code
- Decode Morse Code from files
- Encrypt messages using Caesar Cipher
- Automatically detect Caesar Cipher rotations
